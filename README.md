# Persona - SaaS-платформа для автоматизированного планирования и мотивации пользователей

<b>Persona</b> - это веб-платформа, предоставляющая пользователям инструменты для планирования задач, постановки целей, отслеживания прогресса и повышения мотивации с использованием современных технологий и элементов искусственного интеллекта.

## 🚀 Ключевые возможности 

- <b>Умные заметки и задачи: </b>Создание заметок с вложенным списком задач, отслеживанием прогресса выполнения и дедлайнами.
- <b>Интеграция ИИ: </b>Использование API искусственного интеллекта для генерации задач по указанной теме.
- <b>Социальной взаимодействие: </b>Система подписок на других пользователей для отслеживания их активности, возможность делиться заметками с помощью генерации ссылок.
- <b>Умные уведомления: </b>Система оповещений о приближающихся дедлайнах.
- <b>Категоризация: </b>Система категорий для фильтрации контента.

## 💻 Запуск проекта
Проект полностью контейнезирован. 
### 1. Подготовка
Для запуска проекта необходим [Docker Desktop](https://www.docker.com/products/docker-desktop/).
### 2. Установка
Клонируйте репозиторий, перейдите в папку проекта:
```bash
git clone https://github.com/toejamow/Diploma.git
cd ваша-папка
```
### 3. Запуск
Запустите сборку и подъем контейнеров:
```bash
docker compose up -d --build
```
### 4. Готово
Приложение будет доступно по [ссылке]( http://localhost:5173). Вы великолепны!
